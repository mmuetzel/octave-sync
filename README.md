This repository hosts the workflow that synchronizes the Mercurial repository MXE Octave to GitHub.

MXE Octave contains the build rules for Octave and all its dependencies. It was forked from the [MXE project](https://mxe.cc/) in November 2012.

It is used by the Octave project to cross-build the [Windows installers for GNU Octave](https://www.octave.org/download#ms-windows) on Linux.
